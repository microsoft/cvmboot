cencode
=======

Utility to encode a file as a C data array. For example:

```
$ ./cencode Makefile makefile_data
unsigned char makefile_data[] =
{
    0x61, 0x6c, 0x6c, 0x3a, 0x0a, 0x09, 0x67, 0x63,
    0x63, 0x20, 0x2d, 0x6f, 0x20, 0x63, 0x65, 0x6e,
    0x63, 0x6f, 0x64, 0x65, 0x20, 0x6d, 0x61, 0x69,
    0x6e, 0x2e, 0x63, 0x0a, 0x0a, 0x63, 0x6c, 0x65,
    0x61, 0x6e, 0x3a, 0x0a, 0x09, 0x72, 0x6d, 0x20,
    0x2d, 0x66, 0x20, 0x63, 0x65, 0x6e, 0x63, 0x6f,
    0x64, 0x65, 0x0a, 0x0a, 0x64, 0x69, 0x73, 0x74,
    0x63, 0x6c, 0x65, 0x61, 0x6e, 0x3a, 0x20, 0x63,
    0x6c, 0x65, 0x61, 0x6e, 0x0a, 0x0a, 0x74, 0x65,
    0x73, 0x74, 0x73, 0x3a, 0x0a, 0x09, 0x2e, 0x2f,
    0x63, 0x65, 0x6e, 0x63, 0x6f, 0x64, 0x65, 0x20,
    0x4d, 0x61, 0x6b, 0x65, 0x66, 0x69, 0x6c, 0x65,
    0x20, 0x6d, 0x61, 0x6b, 0x65, 0x66, 0x69, 0x6c,
    0x65, 0x5f, 0x64, 0x61, 0x74, 0x61, 0x0a, 0x00
};
```

An extra zero terminator is appended to the data array.
